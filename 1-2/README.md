# Leave Management System

對應規格：
- specs/001-leave-management/spec.md
- specs/001-leave-management/contracts/openapi.yaml

## 開發環境

- Node.js（建議 Node 20 LTS；本機若使用其他版本，可能需調整前端 Vite 版本）

## Quickstart

詳細請見 specs/001-leave-management/quickstart.md。

常用指令：

- Backend：`npm --prefix backend run start:dev`
- Frontend：`npm --prefix frontend run dev`
- Migrate：`npm --prefix backend run prisma:migrate`
- Seed：`npm --prefix backend run seed`

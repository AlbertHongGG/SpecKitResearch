## Frontend (React + Vite + TypeScript + Tailwind)

UI for the Leave Management system.

### Requirements

- Node.js LTS (Vite may require Node 20.19+ or 22.12+)
- pnpm

### Setup

```bash
pnpm install
cp .env.example .env
```

### Run

```bash
pnpm dev
```

Vite prints the local URL (typically `http://localhost:5173`).

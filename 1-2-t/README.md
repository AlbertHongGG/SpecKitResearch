# Leave Management System

Spec-driven implementation for the Leave Management feature.

- Spec: specs/001-leave-management/spec.md
- Plan: specs/001-leave-management/plan.md
- Tasks: specs/001-leave-management/tasks.md
- Quickstart: specs/001-leave-management/quickstart.md

## Development

- Backend: `cd backend && pnpm install && pnpm start:dev`
- Frontend: `cd frontend && pnpm install && pnpm dev`

If you hit a Vite/Node version warning, upgrade Node to a version supported by the `frontend/` Vite version.

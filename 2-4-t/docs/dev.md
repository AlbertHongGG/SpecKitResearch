# Dev

## Prerequisites

- Node.js 20 LTS
- npm

## Install

```sh
npm install
```

## Run (dev)

```sh
npm run dev:backend
npm run dev:frontend
```

- Backend: http://localhost:3001
- Frontend: http://localhost:3000

## Database

```sh
npm run db:migrate
npm run db:seed
```

## Tests

```sh
npm run test:unit
npm run test:backend
npm run test:frontend
npm run test:e2e
```

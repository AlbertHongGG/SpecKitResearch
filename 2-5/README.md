# Multi-Role Forum

## Quickstart

- Install: `npm install`
- Generate client (optional): `npm run db:generate`
- Setup DB: `npm run db:migrate`
- Seed (optional): `npm run db:seed`
- Dev: `npm run dev`

## Tests

- Unit: `npm test`
- E2E: `npm run test:e2e`

Environment variables: see `.env.example`.
